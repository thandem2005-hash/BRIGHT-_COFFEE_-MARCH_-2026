# BRIGHT-_COFFEE_-MARCH_-2026
📊 Bright Coffee Shop Sales Analysis
📌 Project Overview
This project analyzes historical transactional data from Bright Coffee Shop to generate business insights for a new CEO. The goal is to identify sales trends, top-performing products, and opportunities to improve revenue.
________________________________________
🎯 Objectives
•	Identify top-selling products
•	Analyze revenue by product category
•	Determine peak sales hours
•	Provide recommendations to improve performance
________________________________________
🛠️ Tools & Technologies
•	SQL (Data cleaning & transformation)
•	Databricks / Snowflake (Data processing & storage)
•	Microsoft Excel (Data analysis & dashboards)
•	PowerPoint (Presentation)
•	Miro (Planning & architecture diagram)
________________________________________
🔄 Data Processing Steps
1.	Converted raw data into CSV format
2.	Cleaned data (fixed unit price formatting)
3.	Created calculated column:
total_amount = unit_price × transaction_qty
4.	Extracted time features (hour of transaction)
5.	Grouped data by product and time intervals
________________________________________
📊 Key Insights
•	Certain products generate the highest revenue
•	Peak sales occur during specific hours of the day
•	Some products are underperforming and need promotion
•	Sales patterns show opportunities for growth
________________________________________
📈 Dashboard & Visualizations
The Excel dashboard includes:
•	Revenue by product
•	Sales by hour
•	Quantity sold by category
•	Charts (bar, line, pie)
________________________________________
💡 Recommendations
•	Increase stock of high-performing products
•	Run promotions during slow hours
•	Improve marketing for low-performing items
•	Focus on peak-time sales strategies
